# tree-dir

<b>Simple util to build tree of directory content.</b>

<h3>Usage :</h3> 

    1. Compile "treedir.c" file.
        gcc treedir.c
    
    2. Run the program.
        ./a.out [PATH] [OPTIONS]
        Where options are:
          "-c" for colored output
          "-g" for graphics
          "-h" for help output
